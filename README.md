# VaultKey

**VaultKey** — это надежный и быстрый локальный менеджер паролей для Android, ориентированный на конфиденциальность. Приложение работает полностью автономно: все данные шифруются и хранятся исключительно на вашем устройстве.

## Основные возможности

- **Управление записями:** Хранение логинов, паролей, URL-адресов, заметок и категорий.
- **Безопасный вход:** Доступ по мастер-паролю или с помощью биометрии.
- **Продвинутый генератор:** Создание сложных паролей с настройкой длины и набора символов.
- **Анализ надежности:** Оценка сложности паролей в реальном времени.
- **Озвучка паролей:** Возможность прослушать сгенерированный пароль с регулируемой скоростью.
- **Автоматизация:** Подгрузка иконок сайтов (favicon) по URL.
- **Умный поиск:** Мгновенная фильтрация записей.
- **Кастомизация:** Поддержка светлой и темной тем.

## Безопасность

- **Шифрование:** Используется стандарт AES-256 в режиме GCM.
- **Деривация ключей:** 
  - Основной метод: **PBKDF2 с HMAC-SHA256** (120,000 итераций).
  - Оптимизированный метод (v3): Использование SHA-256 для ускорения работы.
- **Android Keystore:** Безопасное хранение криптографических ключей на аппаратном уровне.
- **Защита данных:**
  - Автоматическая блокировка приложения при бездействии.
  - Очистка буфера обмена.
  - Отсутствие разрешений на доступ в интернет (полный оффлайн).

## Стек технологий

- **Язык:** Java 17
- **UI:** Material Design 3, ViewBinding, Navigation Component
- **База данных:** Room (SQLite)
- **Безопасность:** AndroidX Biometric, JCA
- **Библиотеки:**
  - `Glide` — загрузка иконок.
  - `zxcvbn4j` — оценка сложности.
  - `Gson` — работа с JSON.
  - `Lifecycle` — архитектурные компоненты.

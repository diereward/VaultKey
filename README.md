# VaultKey

**VaultKey** — это надежный и быстрый локальный менеджер паролей для Android, ориентированный на конфиденциальность. Приложение работает полностью автономно: все данные шифруются и хранятся исключительно на вашем устройстве.

## Основные возможности

- **Управление записями:** Хранение логинов, паролей, URL-адресов, заметок и категорий.
- **Безопасный вход:** Доступ по мастер-паролю или с помощью биометрии.
- **Продвинутый генератор:** Создание сложных паролей с настройкой длины и набора символов.
- **Анализ надежности:** Оценка сложности паролей в реальном времени с помощью zxcvbn.
- **Автоматизация:** Подгрузка иконок сайтов (favicon) по URL или использование инициалов.
- **Умный поиск:** Мгновенная фильтрация записей по названию, логину или URL.
- **Кастомизация:** Поддержка системной, светлой и темной тем оформления.

## Безопасность

- **Шифрование:** Используется стандарт AES-256 в режиме GCM.
- **Деривация ключей:** PBKDF2 с HMAC-SHA256 (120,000 итераций) для максимальной защиты от перебора.
- **Android Keystore:** Безопасное хранение мастер-ключа на аппаратном уровне устройства.
- **Защита данных:**
  - Автоматическая блокировка приложения при бездействии.
  - Безусловная очистка буфера обмена через 30 секунд.
  - Отсутствие разрешений на доступ в интернет (полный оффлайн).
  - Валидация URL-адресов.

## Стек технологий

- **Язык:** Java
- **UI:** Material Design 3, ViewBinding, Navigation Component
- **База данных:** Room (SQLite)
- **Безопасность:** AndroidX Biometric, JCA (Java Cryptography Architecture)
- **Библиотеки:**
  - `Glide` — эффективная загрузка и кэширование иконок.
  - `zxcvbn4j` — интеллектуальная оценка сложности паролей.
  - `Gson` — работа с JSON данными.

# VaultKey

**VaultKey** — это надежный и быстрый локальный менеджер паролей для Android, ориентированный на конфиденциальность. Приложение работает полностью автономно: все данные шифруются и хранятся исключительно на вашем устройстве.

## Основные возможности

- **Управление записями:** Хранение логинов, паролей, URL-адресов, заметок и категорий.
- **Безопасный вход:** Доступ по мастер-паролю или с помощью биометрии (отпечаток пальца/Face ID).
- **Продвинутый генератор:** Создание сложных паролей с настройкой длины и набора символов.
- **Анализ надежности:** Оценка сложности паролей в реальном времени с использованием алгоритма `zxcvbn`.
- **Автоматизация:** Подгрузка иконок сайтов (favicon) по URL для удобной навигации.
- **Умный поиск:** Мгновенная фильтрация записей по названию или логину.
- **Кастомизация:** Полная поддержка Material 3, светлой и темной тем.

## Безопасность

- **Шифрование:** Используется стандарт AES-256 в режиме GCM.
- **Деривация ключей:** 
  - Основной метод: **PBKDF2 с HMAC-SHA256** (120,000 итераций) для максимальной защиты от перебора.
  - Оптимизированный метод (v3): Использование SHA-256 для ускорения работы с новыми записями без потери стойкости.
- **Android Keystore:** Безопасное хранение криптографических ключей на аппаратном уровне.
- **Защита данных:**
  - Автоматическая блокировка приложения при бездействии (`AutoLockManager`).
  - Очистка буфера обмена через заданный промежуток времени.
  - Отсутствие разрешений на доступ в интернет (полный оффлайн).

## Стек технологий

- **Язык:** Java 17
- **UI:** Material Design 3, ViewBinding, Navigation Component
- **База данных:** Room (SQLite)
- **Безопасность:** AndroidX Biometric, Java Cryptography Architecture (JCA)
- **Библиотеки:**
  - `Glide` — загрузка и кеширование иконок.
  - `zxcvbn4j` — оценка сложности паролей.
  - `Gson` — работа с JSON данными.
  - `Lifecycle (ViewModel, LiveData)` — реактивное обновление интерфейса.
